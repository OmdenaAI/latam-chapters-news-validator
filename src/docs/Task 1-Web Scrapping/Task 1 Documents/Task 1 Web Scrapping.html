Task 1 Web Scrapping

Web scraping is the process of extracting content and data from websites using software. Web Scrapping is a collection of practices used to automatically extract or “scrape” data from the web. Web Scrapping is an extremely useful tool for online data gathering.

Web scrapping applications include market research, price comparisons, content monitoring and more. It’s possible to scrape all kind of web data.

Reference: (KINSTA, 2022), What Is Web Scraping? How To Legally Extract Web Content
